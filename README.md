## Deploy

### GitHub Pages
1. Push to GitHub
2. Go to Settings → Pages
3. Select `master` branch as source
4. Live at `github-username.github.io/catalyst-landing`

### Netlify
1. Drag `index.html` onto [netlify.com/drop](https://netlify.com/drop)
2. Or connect the repo for auto-deploy

### Cloudflare Pages
1. Connect repo to Cloudflare Pages
2. Auto-deploys on push
